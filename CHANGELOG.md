# Changelog

이 프로젝트의 주요 변경 사항을 기록합니다.

형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/)를 따르며,
버전 체계는 [Semantic Versioning](https://semver.org/lang/ko/)을 따릅니다.
앱 특성상 "호환성"의 기준은 백업 데이터 스키마이며, 코드 버전(`APP_VERSION`)과
데이터 스키마 버전(`DATA_SCHEMA`)은 별도로 관리합니다.

## [2.16.0] - 2026-06-29
### Added
- 모달 키보드 접근성 — `ESC` 키로 최상단 모달 닫기 (아이젠하워·정리 미리보기처럼 상세 모달 위에 겹쳐 열린 경우 위에서부터 차례로 닫힘)
- 모달 열림 중 `Tab`/`Shift+Tab` 포커스 트랩(포커스가 모달 밖으로 새지 않음)과 닫을 때 트리거 요소로 포커스 복귀
- 모든 모달에 `role="dialog"`·`aria-modal="true"` 부여로 스크린리더 인식 개선
### Changed
- `setupModalA11y()` 도입 — 모달 표시 토글(`display:flex/none`)을 `MutationObserver`로 감지해 포커스 이동/복귀를 자동 처리, 전역 `keydown`에서 `ESC`·`Tab` 일괄 관리 (모달이 없을 때는 무동작이라 일반 입력에 영향 없음)

## [2.15.0] - 2026-06-29
### Added
- 모바일/태블릿 칸반 터치 드래그 — 카드를 길게 눌러(롱프레스) 칼럼 간 이동 및 같은 칼럼 내 순서 변경 지원 (네이티브 HTML5 DnD는 터치 이벤트를 발생시키지 않아 모바일에서 동작하지 않던 문제 해결)
- 드래그 중 손가락을 따라오는 카드 복제본과 드롭 위치 인디케이터(파란 선), 칼럼 하이라이트
### Changed
- 카드를 살짝 탭하면 상세 모달이 열리고(클릭 유지), 바로 쓸어내리면 보드가 가로 스크롤되도록 제스처 구분
- 마우스·터치 드롭의 칼럼 이동/순서 변경 로직을 `applyCardMove()` 단일 경로로 통합 (기존 `onCardDrop`/`onColDrop` 중복 제거, 동작 동일성 단위 테스트로 검증)
- 알림 토스트 통일 — 지식 저장 완료·ICS 내보내기에서 각자 만들던 인라인 토스트를 공통 `showToast()`로 일원화 (스타일·동작 일관성 / `showUndoToast`는 되돌리기 버튼이 있는 별도 위젯으로 유지)

## [2.14.1] - 2026-06-29
### Changed
- 성능: 저장(`saveAll`)을 350ms 단위로 모아 기록(debounce)하고, 탭 전환/종료 시 즉시 flush — 상세 모달 편집 중 누적되던 입력 지연 해소
- 성능: 저장공간 게이지 계산이 매 저장마다 `localStorage` 전체를 재스캔하던 비용 제거 — 직렬화한 데이터 문자열 길이 + 캐시된 기타 키 크기로 추정
- 성능: 활동 기록(`history`) 배열에 상한(최근 100개) 적용 — 편집을 반복해도 태스크 객체가 무한정 커지지 않음 (화면 표시는 최근 5개로 동일)
- 활동 기록 렌더링을 반복 `innerHTML +=` 대신 1회 일괄 대입으로 변경

## [2.14.0] - 2026-06-25
### Added
- 사용자 입력값을 화면에 표시하기 전 escape 처리하는 `escapeHtml()` 헬퍼 — 저장형 XSS 방지 (제목·메모·태그·요청자·하위 태스크·AI 생성 콘텐츠 등 약 25곳 적용)
### Fixed
- `localStorage`에 저장된 데이터가 손상되어 JSON 파싱에 실패해도 앱이 멈추지 않고 안내 토스트와 함께 빈 상태로 시작
- 백업 파일 불러오기(`importData`) 시 `tasks`/`wikiItems`/`trashBin` 형식과 태스크 필드 타입을 검증해 잘못된 파일로 인한 오류 방지
- 이미 삭제된 태스크를 대상으로 모달 저장·하위 태스크 추가/토글/삭제·날짜·요청자 수정 시 발생하던 크래시 방지 (`tasks.find()` 결과 null 가드 추가)
- 매월 반복 태스크의 기준일이 31일처럼 다음 달에 존재하지 않는 날짜일 때 한 달을 건너뛰던 문제 수정 (다음 달 말일로 보정)
